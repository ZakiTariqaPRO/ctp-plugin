# Captures d'écran de la page d'installation

Dépose ici tes captures (PNG/JPG), puis remplace les blocs placeholder dans `docs/index.html`.

## Comment remplacer un placeholder par une vraie image

Dans `docs/index.html`, chaque emplacement ressemble à :

```html
<div class="shot">📷 Capture à ajouter : ...</div>
```

Remplace-le par :

```html
<div class="shot"><img src="img/NOM-DE-TA-CAPTURE.png" alt="description"></div>
```
