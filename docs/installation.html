Pourquoi Git ? « Git » est un petit outil gratuit qui permet à Claude d'aller chercher la boîte à outils du Collectif. Sans lui, l'installation automatique ne se fait pas. C'est à installer une seule fois.

Pour Mac 🍎

Mac — installation pas à pas

1

Installe l'application Claude

  • Va sur claude.ai/download et clique Download for Mac.
  • Ouvre le fichier téléchargé (.dmg) et glisse l'icône Claude dans le dossier Applications.
  • Ouvre Claude depuis tes Applications et connecte-toi.
📷 Capture : page claude.ai/download (Mac) + glisser Claude dans Applications
2

Installe Git (les « outils ligne de commande »)

Sur Mac, Git s'installe via les « outils de développement en ligne de commande » d'Apple. Deux méthodes — prends la plus simple :

Méthode A (la plus simple, sans rien taper) :

  • Quand Claude (ou Mac) a besoin de Git, une fenêtre apparaît : « L'outil "git" nécessite les outils de développement en ligne de commande ».
  • Clique Installer, accepte, et attends la fin (quelques minutes).

Méthode B (si la fenêtre n'apparaît pas) :

  • Ouvre l'app Terminal (cherche « Terminal » avec ⌘ + Espace).
  • Copie-colle cette ligne, puis appuie sur Entrée :

xcode-select --install

  • Une fenêtre s'ouvre → clique Installer et attends la fin.
Pas besoin de HomebrewTu n'as PAS besoin d'installer Homebrew ni quoi que ce soit d'autre. Les outils Apple suffisent.

Pour Windows 🪟

Windows — installation pas à pas

1

Installe Git pour Windows (à faire EN PREMIER)

  • Va sur git-scm.com/downloads/win et télécharge Git for Windows (64-bit dans la majorité des cas).
  • Ouvre le fichier téléchargé.
  • Clique Next / Suivant à chaque écran sans rien changer (laisse toutes les options par défaut), puis Install.
  • Attends la fin, puis clique Finish.
Ne saute pas cette étapeSans Git, la boîte à outils du Collectif ne pourra pas s'installer. C'est rapide et gratuit.
📷 Capture : page git-scm.com/downloads/win + écran « Next/Install » de l'installateur
2

Installe l'application Claude

  • Va sur claude.ai/download et clique Download for Windows.
  • Ouvre le fichier téléchargé et installe-le (clique « Suivant » jusqu'au bout).
  • Lance Claude depuis le menu Démarrer et connecte-toi.
Pas besoin de WSLL'application Claude fonctionne directement sur Windows. Tu n'as rien d'autre à installer.
