## Règles de remplissage
- Chaque fait : **source** (interview / persona / offre / observation) + **date**.
- **Énoncé Moore sans case vide** — une case floue = à retravailler, pas à publier.
- Différenciateur non prouvé / copiable → étiqueté hypothèse, pas présenté comme acquis.
- Pas de chiffres d'argent dans les sorties publiques.
- Contradiction avec une version antérieure → archiver l'ancienne en note datée.
