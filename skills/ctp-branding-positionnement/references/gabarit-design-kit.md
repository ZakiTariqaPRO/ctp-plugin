## Règles
- Codes hex obligatoires (réutilisables direct dans Canva, Figma, etc.).
- Polices courantes/gratuites uniquement (le membre doit pouvoir les avoir).
- Cohérence avec les associations de marque, jamais l'inverse.
- Si le projet n'est pas le Collectif Tariqa PRO, ne PAS imposer beige/noir/or.
- Pour des planches visuelles / logo : passer la main au skill `ctp-brandkit` (bundlé).
